# ClubHub
 
> All-in-one club management system — member & season pass management, NFC-based point-of-sale, and gate entry control for sports clubs.
 
ClubHub is a self-hostable platform built for sports clubs of any size. It combines member administration, season pass issuance with NFC card support, a point-of-sale system for canteen operations, an online shop for fans, and a gate entry control module — all in a single application.
 
---
 
## Features
 
- **Member Management** — Create and manage club members, assign season passes, generate NFC keys
- **Season Pass & NFC Cards** — Issue physical NFC cards (NTAG213/215) with embedded keys; includes configurable free drink allowances based on the amount paid
- **Point of Sale (POS)** — Canteen cashier system with product catalog, shopping cart, and NFC card scanning to redeem free drinks automatically
- **Gate Entry Control** — Scan NFC cards at the entrance to verify pass validity in real time; green/red visual feedback with member name display
- **Online Shop** — Fans can order and pay for season passes online (Stripe); pickup via generated code at the club office
- **Revenue Reports** — Track sales by product, date range, and cashier session
